# Spotify Account Checker CLI

----

### Features

* Subscription ( Premium, Premium For Family, Free)
* Username Account
* Country
* Expired Date (DD/MM/YYYY)

----

### Installation

```
$ git clone https://github.com/mbul48/spotify.git
$ cd spotify
$ php spotify.php
```
